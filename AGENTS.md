# 开发核心

一个自动有声PPT视频的项目(Banana Lecture)，具体呈现如下工作流，用户输入相关提示词，大模型首先根据相关内容生成每一页的规划，然后根据每一页的规划生成每页的对应的对话口播稿，然后再生成每一页的图片，对话口播稿生成对应的音频，最后将图片与音频组合成视频输出，提供最终的下载链接

## 核心组件

### 侧边栏

- Logo呈现
- 项目管理
- 配置管理

### 规划空间

- 用户输入框
- 对话式修改PPT规划
- 手动修改PPT规划具体内容
- 跳转到工作空间

### 工作空间

- 整体进度条显示
- 快捷按钮配置(一键生成、一键导出),具体按钮配置(一键生成口播稿,一键生成图片)
- 展示每一页的图片与对话口播稿
- 支持手动修改口播稿内容
- 支持对话修改图片内容与口播稿内容
- 返回上一页/下一页(页数与规划的页数一致)
- 返回规划空间

# 开发要点
- 每次结束前, 调用`npm run build`验证代码是否编译通过
